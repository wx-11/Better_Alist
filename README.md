## 使用方法
   * 以下内容针对静态背景，视频背景详见[此处](https://github.com/wx-11/Better_Alist/issues/7)
   * 在Alist后台页面依次选择```设置```-->```全局```
   * 将以下内容添加到对应框框内
     * 自定义头部
        ```html
        <link href="https://cdn.jsdelivr.net/gh/wx-11/Better_Alist@main/alist.css" rel="stylesheet" type="text/css">
        ```

     * 自定义内容

        ```html
        <script src="https://cdn.jsdelivr.net/gh/wx-11/Better_Alist@main/jq.js"></script>
        <div id="jsi-flying-fish-container" class="fish-container"></div>
        <script src='https://cdn.jsdelivr.net/gh/wx-11/Better_Alist@main/fish.js'></script>
        ```
